LROC Magazine handover from v3.4.33

Current baseline: v3.4.33 — Advert/Image Asset Intent Controls.

Important behaviours to preserve:
- Article A4 preview and Issue Builder use the same page-window renderer and phantom-page trimming from v3.4.28.
- Article image captions work by double-click / Caption selected image.
- Insert Advert has no captions, no article image border, and uses advert layout controls.
- Insert Advert now accepts any image asset because some artwork arrives as normal images but is semantically an advert.
- Advert Library image import creates an advert item from the uploaded image and auto-senses basic shape/orientation.
- Normal article images support full, half, left wrap, right wrap and reset layout controls.
- Deploy helper v1.13 remains in lroc/deployer/deploy_helper.py.

Next likely work:

- Tune half/wrap widths if the preview differs from editor expectations.
