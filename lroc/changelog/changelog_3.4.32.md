# LROC v3.4.32 — Inline Advert Fill Consistency

Timestamp: 2026-06-16 22:27:30 AEST

## Changes

- Inline adverts now default to full-column layout when inserted through **Insert advert**.
- A4 preview and Issue Builder now make the advert artwork fill the selected advert figure, matching the TinyMCE editor more closely.
- Existing inline adverts with no stored advert layout are normalised to full-column layout on save/preview.
