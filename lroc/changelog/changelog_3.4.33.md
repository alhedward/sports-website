# LROC Magazine v3.4.33 — Advert/Image Asset Intent Controls

- Insert Advert can now use any image asset, not only generated advert assets.
- Advert Library image import now treats the uploaded image as finished advert artwork instead of reusing a normal image asset.
- Advert image imports auto-detect basic shape/orientation and preselect size/orientation fields.
- Article image layout controls now match advert controls more closely with full, half, left wrap, right wrap and reset.
